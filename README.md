# EvanGoldGrid - MT5 网格交易系统

## 概述

EvanGoldGrid 是一款专业的 MetaTrader 5 (MT5)  Expert Advisor (EA)，采用网格交易策略，支持多种交易模式和智能风险控制功能。适用于黄金 (XAUUSD) 及其他金融产品的自动化交易。

**版本**: 1.53  
**平台**: MetaTrader 5  
**语言**: MQL5

---

## 核心特性

### 1. 四种交易模式

| 模式 | 说明 | 订单类型 |
|------|------|----------|
| **做多 (Long)** | 所有网格均为买单 | BUY_LIMIT / BUY_STOP |
| **做空 (Short)** | 所有网格均为卖单 | SELL_LIMIT / SELL_STOP |
| **多空交替 (Alternating)** | 买单和卖单交替分布 | 混合订单 |
| **双边网格 (Dual Side)** | 每个网格同时有买单和卖单 | 双向订单 |

### 2. 智能网格系统

- **中心价格**: 可手动设置或实时刷新为当前市场价格
- **网格数量**: 支持最多 100 个网格
- **网格间距**: 可自定义价格间隔 (点数)
- **自动补单**: 订单成交后自动在相同网格位置重新挂单

### 3. 风险控制

- **最大亏损保护**: 设置最大亏损金额 (USD)，达到阈值自动停止交易并平仓
- **盈利保护**: 当盈利达到阈值后触发保护机制，盈利回撤时自动平仓
- **订单数量限制**: 限制最大同时存在的订单数量
- **交易时段控制**: 可设置交易小时和允许的交易日

### 4. 交互式控制面板

实时显示交易状态，支持一键操作：
- 生成/重新生成网格
- 平亏损仓位
- 平盈利仓位
- 全部平仓
- 撤销所有挂单
- 撤销挂单 + 平仓
- 切换自动补单
- 切换盈利保护
- 刷新中心价格


### 2. 安装步骤

1. 将 `EvanGoldGrid.mq5` 和 `CEvanGoldGrid.mqh` 复制到 MT5 数据目录：
   ```
   <MT5 数据目录>\MQL5\Experts\EvanGoldGrid\
   ```

2. 在 MT5 中打开 MetaEditor (F4)，编译项目

3. 在 MT5 图表上添加 EA：
   - 导航到"导航器" → "智能交易系统" → "EvanGoldGrid"
   - 拖拽到目标图表

- MetaTrader 5 平台
- 允许 EA 交易 (工具 → 选项 → EA → 允许自动化交易)
- 允许实时事件 (图表属性 → 允许图表事件)

## 交易逻辑详解

### 网格订单生成

1. **价格计算**:
   ```
   起始偏移 = -((网格数量 - 1) / 2) × 网格间距
   网格价格[i] = 中心价格 + 起始偏移 + (i × 网格间距)
   ```

2. **订单类型判断**:
   - **买单**:
     - 网格价格 < 当前问价 → BUY_LIMIT
     - 网格价格 >= 当前问价 → BUY_STOP
   - **卖单**:
     - 网格价格 > 当前报价 → SELL_LIMIT
     - 网格价格 <= 当前报价 → SELL_STOP

3. **双边模式**: 每个网格同时创建买单和卖单

### 自动补单机制

1. EA 维护一个网格缓存，记录每个网格的目标价格和方向
2. 每 0.1 秒检查一次网格状态
3. 当发现网格中没有订单 (挂单或持仓) 时，自动补单
4. 手动干预操作后会暂停自动补单

### 盈利保护机制

1. **动态激活阈值**: 网格数量 × 网格间距 × 2.0
2. **动态触发值**: 网格数量 × 网格间距 × 0.5
3. 当盈利达到激活阈值时，保护机制启动
4. 当盈利回撤到触发值以下时，自动停止交易并平仓

### 止损机制

- 实时监控账户净值
- 当亏损达到 `InpMaxLossUSD` 时：
  - 停止所有交易
  - 撤销所有挂单
  - 关闭所有持仓
  - 禁用自动补单

## 订单命名规则

所有订单和持仓都带有注释标签，格式为：
```
Grid_<网格索引>_<订单类型>
```

示例：
- `Grid_0_BUY_LIMIT` - 第 0 号网格的限价买单
- `Grid_5_SELL_STOP` - 第 5 号网格的止损卖单

## 注意事项

### 风险提示

1. **网格交易风险**: 网格策略在单边行情中可能产生较大浮亏
2. **资金管理**: 建议设置合理的最大亏损限额
3. **杠杆风险**: 高杠杆可能放大亏损
4. **市场风险**: 重大新闻事件可能导致滑点或跳空

### 最佳实践

1. **回测**: 在实盘前进行充分的历史回测
2. **模拟盘**: 先在模拟账户运行至少 2 周
3. **小仓位**: 初始使用最小手数测试
4. **监控**: 定期监控 EA 运行状态
5. **备份**: 定期备份交易记录和设置

### 技术限制

1. **订单数量**: 受经纪商最大订单数量限制
2. **交易时间**: 周末和节假日市场休市时无法交易
3. **网络依赖**: 需要稳定的网络连接
4. **VPS 推荐**: 建议 24/7 运行在 VPS 上

## 故障排除

### 常见问题

**Q: EA 不生成订单？**
- 检查是否允许 EA 交易
- 确认在交易时间内
- 检查订单数量是否已达上限
- 验证账户余额是否足够

**Q: 自动补单不工作？**
- 确认自动补单功能已开启
- 检查是否进行了手动干预操作
- 验证网格缓存是否有效

**Q: 面板不显示？**
- 重启 MT5 平台
- 重新加载 EA
- 检查图表属性设置

**Q: 订单被拒绝？**
- 检查价格是否超出市场范围
- 验证手数是否符合经纪商要求
- 确认账户类型允许挂单

### 日志查看

EA 运行日志输出到 MT5"专家"标签页，关键信息包括：
- 网格生成详情
- 订单成交/失败信息
- 自动补单操作
- 风险控制触发
